# Stratégie MVP (Minimum Viable Product)

L'objectif du MVP est de fournir un moteur fonctionnel pour gérer des produits physiques et digitaux simples dans un environnement Next.js.

## Fonctionnalités Critiques (Inclus dans le MVP)
1.  **Core** : Plugin Manager basique, Event Bus local.
2.  **Product** : CRUD Produits + Variantes.
3.  **Pricing** : Gestion simple des prix (Prix unique).
4.  **Inventory** : Gestion des stocks par variante.
5.  **Admin UI** : Liste des produits, Formulaire de création simple.
6.  **Next.js** : Helpers pour afficher un produit et gérer le stock via Server Actions.

## Repoussé après la v1
- Marketplace de plugins publique.
- Import/Export CSV complexe.
- Analytics avancés.
- IA pour la génération de descriptions.
- Support multi-langues complet (Internationalization).
